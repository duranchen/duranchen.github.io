使用sql进行CRUD操作（MySQL Version）

Posted on 2016-04-16   |   In Method

示例数据库：sakila database，详细介绍。

所谓CRUD操作是指对数据库的数据进行Create，Read，Update，Delete操作，下面我们一一介绍：

  1. Create

    新建数据，即插入数据。

  2. Read

    读取数据，即查询数据。

    除此之外，还有多表查询和子查询。

  3. Update

    更新数据。

  4. Delete
    删除数据。
